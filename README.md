# ✨ Omicronde Editor ✨
The Official Omicronde Editor, by [Galaxaris](https://github.com/galaxaris/).

_**Omicronde Editor**_ is a powerful and easy-to-use editor for creating and editing games based on the Omicronde API. It provides a user-friendly interface for designing levels, managing assets, and configuring game settings.

_Developed by Galaxaris_ 


**Still in development (Beta)! The editor is currently not fully working.**
